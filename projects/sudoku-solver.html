Sudoku Solver

Development of a modular Sudoku solver able to read a grid from a user-provided JPG image, recognize each digit, solve the puzzle and generate the final resolved output automatically. This project focused on the automatic resolution of a 9×9 Sudoku grid from a JPG image provided by the user. The workflow combines LabVIEW for image processing, a C executable for digit recognition, and MATLAB for solving and generating the final result.

Key Contributions

  • Image reading and grid extraction in LabVIEW
  • Cell-by-cell digit recognition through a C module
  • Integration of a MATLAB solving script
  • Connection of the full software pipeline
  • Automatic generation of the solved output

System Architecture

The project was built as a multi-software pipeline in which each environment had a dedicated role, from image preprocessing to final Sudoku resolution.

This structure made it possible to separate image analysis, digit recognition and solving logic while ensuring reliable data transfer between the different modules.

Workflow

The input image is checked, converted and split into 81 cells. Each cell is then analyzed to identify the corresponding digit.

Once the grid is reconstructed, the MATLAB solver computes the solution and the final result is exported through the main interface.

Technical Stack

  • LabVIEW for orchestration and image handling
  • C executable for OCR-style digit recognition
  • MATLAB for Sudoku solving logic
  • Automated data exchange between modules
  • Final output generation in a complete pipeline
